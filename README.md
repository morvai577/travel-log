### What data are we storing?
- Main entity is `Log entry`
- Each `Log entry` contains the following information
    - Title - Text
    - Description - Text
    - Comments - Text
    - Rating - scale of 1 - 10
    - Image - Text - URL
    - Start date - DateTime
    - End date - DateTime
    - Latitude - Number
    - Longitude - Number
    - Created At - DateTime
    - Updated At - DateTime
